# Fonts Module

This module provides classes for rendering text and emojis.

Fonts are **TTF** files available on the developer’s machine.
They must be prepared using the free **BMFont** tool:
https://www.angelcode.com/products/bmfont/

Each font used in the **Monad(a) Engine** can be rendered in plain, *italic*, or **bold** style
(4 variants in total).
Each font is accompanied by an XML file generated by BMFont.
This XML format is simplified by the engine during asset building.

Each font also requires a texture atlas containing all glyphs.
The engine does not generate this atlas automatically — it must be created
using an external tool or online service.

Emojis are rendered as square glyphs with a size similar to the current font height.
They are based on assets from the **Noto Emoji** project.
Emojis are mixed with text using textual identifiers rather than character codes.
Notably, any resource view already loaded by the application can be used as an emoji.

Meshes used for rendering text and emojis can be replaced at runtime,
for example when the application changes its UI language.
