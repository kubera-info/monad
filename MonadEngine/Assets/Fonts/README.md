# Introduction to fonts in Monad(a) Engine
### The Monad(a) Engine's assets
In the engine all fonts has been created by the BMFont app.
All metadata are in **XML** format.
The font texture is in **DDS** format.
Each font can be bold or italic or plain.
There are **four** **XMLes** and **textures** for each font.
Finnaly there are consolidated into one file.
**XMLes** are also altered to simplify and shornened attributes and elements.
Each style has a fixed position in the atlas.
Every **font** is defined the same way and can be used by the same **classes for fonts**.
